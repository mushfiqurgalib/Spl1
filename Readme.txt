This program parses an html file and it will show any tag's parent,child,sibiling.Also it will count total tag numbers.It can also show latex conversion for an html file.To see latex conversion of empty html tags such as Image,link press 4 and enter the tag name like <img> . To see latex conversion of other html tags, press 5 and enter tag name like <p>.At end,you have to press 6 to exit.Then if we get to the .tex file we will get our latex version of html tag.
